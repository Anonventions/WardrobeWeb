Usage

The plugin provides various features to customize player skins. Here's how you can use the plugin:

  • To create more skins that work with jackets, please head to the 'Shirts _scl Creator' where it'll cut out the second-layers of your shirts automatically.
  • To upload more skin parts, please keep in mind that you need to place them within their corresponding sub-folders to keep it clean and easier to set-up.
  • Use the command /clothes reload to reload the plugin.
  • Use the command /clothes add [overlay_name] [target_skin] to add an overlay to a skin.
  • Use the command /clothes remove [overlay_name] [target_skin] to remove an overlay from a skin.
  • Use the command /clothes reset to reset all skins and overlays.

Replace [overlay_name] with the name of the overlay you want to add or remove, and replace [target_skin] with the target skin name.

Commands

List of available commands:

  • /clothes reload - Reloads the plugin.
  • /clothes add [overlay_name] [target_skin] - Adds an overlay to a skin.
  • /clothes remove [overlay_name] [target_skin] - Removes an overlay from a skin.
  • /clothes reset - Resets all skins and overlays.

Replace [overlay_name] with the name of the overlay you want to add or remove, and replace [target_skin] with the target skin name.

Make sure to have the necessary permissions to use these commands.

Permissions

The plugin provides the following permissions:

  • wardrobeclothes.use - Allows usage of the plugin commands.
  • wardrobeclothes.reload - Allows reloading the plugin.
  • wardrobeclothes.overlay.[overlay_name] - Allows adding or removing a specific overlay.

Replace [overlay_name] with the name of the overlay you want to grant permission for.
